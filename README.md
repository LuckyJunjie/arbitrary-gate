## 🎯 项目定位

「时光笺」是一款 AI 驱动的互动叙事游戏，让用户以"穿越者"身份进入历史现场，通过抽卡、选择、扮演，最终生成一部属于自己的短篇小说。

**核心差异**：不是让你"观看"历史，而是让你在历史的裂缝里，拥有一具身体，一双眼睛，和几次选择的机会。
