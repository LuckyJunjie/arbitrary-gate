## 一、卡牌基础规格

| 属性 | 说明 |
|------|------|
| 卡牌编号 | KW-XXX（关键词）/ EV-XXX（历史事件） |
| 稀有度 | 凡品 / 珍品 / 奇品 / 绝品 |
| 卡牌类型 | 器物 / 职人 / 风物 / 情绪 / 称谓 / 历史事件 |
| 故事标签 | 关联的主题词（1-3个） |
| 扩展包 | 默认为1，后续扩展递增 |

---

## 二、抽取权重与稀有度

### 稀有度定义

| 稀有度 | 抽取概率 | 说明 | 示例 |
|--------|----------|------|------|
| 凡品 | 60% | 普通常见物件/情绪/职人 | 旧船票、煤油灯、货郎 |
| 珍品 | 25% | 有故事性的物品/知名典故 | 玉佩、霸王别姬、孟姜女 |
| 奇品 | 12% | 极有画面感的器物/重大历史时刻 | 赤壁东风、破釜沉舟、空城计 |
| 绝品 | 3% | 标志性物品（如"破釜沉舟"里的"釜"，"鸿门宴"里的"玉璧"） | 霸王剑、诏书、玉璧 |

### 抽取权重示例

```
基础十连抽：
- 凡品：6张
- 珍品：2.5张（向上取整3张）
- 奇品：1.2张（向上取整1张）
- 绝品：0.3张（概率触发）

保底机制：每50抽必出一张奇品及以上
```
