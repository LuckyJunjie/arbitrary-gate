# 时光笺：一场可以走入的历史故事

## 一、产品核心理念

想象你推开一扇门，门后是咸阳宫外那个决定命运的黄昏——子婴捧着玉玺，素车白马，走向刘邦的大营。你不是在读这段历史，你**就是**子婴身边那个拿不定主意的近臣。

或者，你只是渭水边一个卖酒的妇人，那天黄昏，你看见一队素车白马从宫门驶出，整个咸阳城安静得不正常。你放下酒勺，跟了上去。

这就是「时光笺」想做的事：**用三个关键词，锚定一个历史现场，然后把你扔进去。**

不是让你"观看"历史，而是让你在历史的裂缝里，拥有一具身体，一双眼睛，和几次选择的机会。

每一次选择，都是一次"如果"——如果当年赤壁那天东风没有来？如果马嵬驿那夜有人提前报信？如果玄武门之变那天早晨，李建成没有走那条路？

**历史没有如果。但故事有。**

这就是「时光笺」的核心：用AI Agent驱动的互动叙事，让用户以"当事人"或"旁观者"的身份进入历史现场，在关键词的指引下，走出属于自己的故事线，最终获得一部完整的、文学品质的短篇小说。

## 二、核心玩法：抽卡·入局·落笔

### 2.1 抽卡机制

每次开始一段新故事前，用户需要抽取三张【关键词卡】和一张【历史事件卡】。

**关键词卡池** 来自1000个名词关键字，分为：
- **器物**（如：旧船票、铜锁芯、搪瓷缸）
- **职人**（如：说书匠、锔瓷匠、摆渡人）
- **风物**（如：青石板、风雨桥、渡口船）
- **情绪**（如：意难平、乍见欢、舍不得）
- **称谓**（如：娘亲舅、独苗苗、掌上珍）

每张卡有稀有度（凡·珍·奇·绝），稀有度影响该关键词在故事中的"权重"。

**历史事件卡池** 来自600+历史转折点，包括但不限于：
- 巨鹿城·破釜沉舟
- 赤壁崖·东风骤起
- 马嵬驿·杨贵妃缢死
- 陈桥驿·黄袍加身
- 崖山海·十万投海
- 玄武门·李世民射兄
- 断桥·残雪重逢（虚构类，来自白蛇传）

**抽取规则**：每日免费抽3次，可消耗"时光笺"道具额外抽取。用户可以保留手牌，最多持有9张关键词卡和3张事件卡。

### 2.2 入局：角色生成

选定三张关键词卡和一张事件卡后，系统进入【入局】阶段。

**第一步：选择身份视角**

每个历史事件提供三种预设身份视角，用户择一进入：

| 视角类型 | 示例（马嵬驿） | 体验特点 |
|---|---|---|
| **局中人·高位** | 高力士身边的小太监 | 离权力核心最近，能看到最多内幕，但选择风险最大 |
| **局中人·低位** | 马嵬驿的驿卒 | 底层视角，只能看到历史洪流的一角，但行动相对自由 |
| **旁观者** | 驿道边茶摊的老妪 | 完全的外部视角，无法改变历史主线的结局，但可以改变具体人物的命运 |

**第二步：配角群像生成**

AI根据所选事件和三张关键词，生成6-8个配角，分为：
- **命运羁绊者**（1-2人）：与主角有强烈情感或利益关联
- **历史节点人物**（2-3人）：真实历史人物，大命运不可改变，但细节可被重新诠释
- **市井过客**（2-3人）：普通百姓，其小命运完全开放

**第三步：关键词落位**

三张关键词被AI分配到故事的不同位置：核心意象、转折道具、人物关联。

### 2.3 叙事推进：场景选项与分支

故事以"章节"为单位推进，5-8个章节。

**每个章节的结构：**
1. **场景渲染**：AI以文学化语言描写当前场景，融入关键词意象（200-300字）
2. **情境困境**：主角面临一个具体的选择时刻
3. **分支选项**：提供3个选项，每个选项对应不同的价值取向
4. **选择落定**：用户选择后，AI即时生成场景推进（150-200字）
5. **涟漪提示**：屏幕边缘浮现淡灰色字，提示配角命运或关键词状态变化

**隐藏机制：**
- **关键词共鸣**：用户选择与关键词"精神气质"高度一致时，触发"共鸣"
- **历史偏离度**：记录用户选择与真实历史走向的偏离程度
- **配角命运值**：每个配角有隐藏的命运值，受用户选择影响

### 2.4 落笔：短篇小说生成

故事完结后，系统进入【落笔】阶段。

**生成内容：**
1. **标题**：AI生成3个备选标题
2. **题记**：一句散文诗式的引子
3. **正文**：完整的短篇小说，3000-8000字
4. **后日谈**：以"史官曰"或"说书人曰"的口吻，简短点评

**风格选项**：白描 / 江湖 / 笔记 / 话本

## 三、AI Agent角色设计

| Agent名称 | 职责 | 人格设定 |
|---|---|---|
| **说书人** | 场景描写、叙事推进、小说成文 | 温和的中年文人，擅用白描 |
| **判官** | 生成选项、评估选择后果、计算命运偏离度 | 冷面但公正，故意设置道德两难 |
| **画师** | 为关键词卡、场景卡生成配图 | 水墨淡彩+浮世绘版画感 |
| **稗官** | 后日谈，配角命运判词 | 野史爱好者口吻 |
| **掌眼** | 检查文学质感，剔除AI腔 | 严苛的老编辑 |

## 四、与市面上AI互动叙事产品的差异

| 对比维度 | 现有AI互动叙事产品 | 时光笺 |
|---|---|---|
| 起点 | 用户输入一句话或选择模板 | 抽卡+历史事件，起点有仪式感和随机趣味 |
| 叙事主体 | AI推动，用户被动选择 | 用户选择驱动，多AI Agent协作服务 |
| 文学品质 | 普遍偏AI腔，缺乏风格 | 内置文学风格选项，有"掌眼"Agent把关 |
| 成稿价值 | 多为聊天记录式碎片 | 生成完整短篇小说，具有收藏和分享价值 |
| 世界观 | 架空或通用 | 基于真实历史事件，有文化厚度 |

## 五、一句话总结

> **「时光笺」不是让你"玩"一个故事，而是让你在历史的裂缝里，替那些没有名字的人，活一次，选一次，留下一部属于你自己的短篇小说。**
