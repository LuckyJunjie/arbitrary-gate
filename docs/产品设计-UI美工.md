# 时光笺 · UI美工设计文档

## 一、总体美术风格

### 1.1 核心意象

**一间旧时文人的书房，黄昏的光从雕花窗格漏进来。**

不是金碧辉煌的宫殿，不是寒酸破败的茅屋。是一间恰到好处的书房——有书卷气，也有烟火气；有岁月的包浆，但没有陈腐的味道。

### 1.2 色彩体系

| 色彩 | 色值 | 用途 |
|---|---|---|
| **墨色** | #2C2C2A | 主文字、图标线条、边框 |
| **牙色** | #F5F0E6 | 主背景（宣纸底色） |
| **赭石** | #8B5E3C | 强调色（印鉴、朱批、稀有标识） |
| **黛青** | #4A6B6B | 辅助色（墨池、水元素、涟漪） |
| **泥金** | #C9A96E | 点缀色（绝品卡框、特殊成就） |
| **霜白** | #E8E0D5 | 卡片底色、卷轴底色 |
| **烟灰** | #6B6B6B | 次要文字、涟漪提示、批注 |

整体色调偏暖、偏灰、偏低饱和。拒绝"古风游戏"常见的大红大金、高饱和配色。要的是**沉静**，不是热闹。

### 1.3 字体规范

| 用途 | 字体 | 说明 |
|---|---|---|
| 标题 | 思源宋体 CN Heavy | 刻本风格，有分量 |
| 正文 | 思源宋体 CN Regular | 阅读舒适，有书卷气 |
| 批注/涟漪 | 方正清刻本悦宋 | 手写感，略潦草 |
| 选项/按钮 | 思源宋体 CN Medium | 朱红色，略大于正文 |
| 数字/标签 | 思源黑体 CN Light | 干净，不抢戏 |

### 1.4 贯穿元素

- **墨迹**：不规则边缘、晕染效果、浓淡变化
- **纸纹**：宣纸的细微纤维纹理，不是纯色背景
- **光影**：窗格投影、烛光摇曳、墨池反光
- **缺角**：卡片、手稿、分享图都有"残缺"的设计母题

## 二、主界面：书房

### 2.1 整体布局

```
┌─────────────────────────────────────────┐
│  [窗格光影]                               │
│                                          │
│     ┌──────────┐    ┌────────────────┐  │
│     │  墨池    │    │   卷轴预览     │  │
│     │  (抽卡)  │    │  (当前故事)    │  │
│     └──────────┘    └────────────────┘  │
│                                          │
│     ┌──────────┐    ┌────────────────┐  │
│     │  卡匣    │    │   书架         │  │
│     │          │    │                │  │
│     └──────────┘    └────────────────┘  │
│  ──────────────────────────────────────  │
│  [书案台面·木质纹理]                       │
└─────────────────────────────────────────┘
```

### 2.2 墨池（抽卡入口）

**位置**：主界面左上方，最显眼的"镇纸"位置。

**视觉**：一方石砚，椭圆形，池中有半池墨。砚台边缘有使用的磨损痕迹——不是崭新的器物，是用了很久的老物件。

**动态效果**：
- 待抽卡时，墨面平静，偶尔泛起极细微的涟漪（呼吸感）
- 可抽卡时，墨面中心浮现一点金光（泥金色），缓慢扩散
- 点击后，墨面涟漪大起，卡片从墨中"浮出"

**墨迹占卜提示**：墨池上方，悬空浮现一行极淡的烟灰色小字：
> *今日墨色偏青，似有旧物来寻。*

### 2.3 书架

**视觉**：一个老式书架，竖放着若干本"书"。每本书的厚度不同——故事越长，书越厚。书脊上贴着题签，写有故事标题（手写楷体，墨色深浅不一）。已读完的书，书脊底部有一个小小的朱红印鉴。

### 2.4 光影氛围

主界面有一层淡淡的窗格投影，根据手机时间，投影的角度和色温会变化：
- 早晨：投影偏长，色温偏冷（黛青调）
- 正午：投影最短，色温中性
- 黄昏：投影偏长，色温偏暖（赭石调）
- 夜间：窗格影消失，取而代之的是烛光摇曳的效果

## 三、抽卡界面：墨池

### 3.1 残片拼接交互

当用户点击墨池抽卡时：
1. 墨面涟漪大起
2. 卡片的一角从墨中刺出（只露出约1/5）
3. 屏幕下方浮现一行字：*「墨中有物，拂去墨迹」*
4. 用户需要在屏幕上做"擦拭"手势（手指在卡片露出处来回滑动）
5. 随着擦拭，墨迹被"擦掉"，卡片逐渐完整显现
6. 卡片完全显现后，墨迹从卡片表面滑落回墨池（有墨汁滴落的小动画）

### 3.2 卡片展示

**卡片正面**：
- 底色：霜白宣纸纹理
- 边框：根据稀有度变化（凡品无框，珍品洒金细框，奇品绢帛纹框，绝品泥金细框+玉版宣底纹）
- 主图：手绘风格的小画，占卡片上半部（约3/5）
- 关键词：三个大字，居中，楷体，墨色（绝品为泥金色）
- 稀有度标记：卡片右下角有一方小印，印文为"凡/珍/奇/绝"（篆书）

## 四、故事界面：卷轴

### 4.1 卷轴布局

故事界面是一幅竖向展开的卷轴，占满全屏。

```
┌─────────────────────────────────────────┐
│ ═══════════════════════════════════════  │ ← 天杆(木质)
│   [进度墨线]                              │
│                                          │
│   正文区域(竖向排版，从右向左书写)            │
│   那天的渡口起了大风。                      │
│   芦苇被吹得伏倒在地，                      │
│   江面上翻着白浪。                          │
│   你正要收缆回棚，                          │
│   对岸来了个人。                            │
│                                          │
│   ┌────────────────────────┐            │
│   │ ○ 退回马厩，藏起来       │            │
│   │ ○ 硬着头皮往前走         │            │
│   │ ○ 转身绕道后院          │            │
│   └────────────────────────┘            │
│   [涟漪区: 淡灰小字]                       │
│ ═══════════════════════════════════════  │ ← 地杆(木质)
└─────────────────────────────────────────┘
```

### 4.2 手势交互区域

当故事进入关键选择时刻，文字选项消失，取而代之的是手势引导：
- ← 向后滑 → 向前推
- ↻ 画一个圈

用户做出手势后，界面有即时的触感反馈（轻微震动）。

## 五、成书界面：手稿

### 5.1 手稿正文

**排版**：
- 竖向卷轴式阅读，可上下滑动
- 字体：手写楷书风格，有轻微的笔画粗细变化
- 墨色：正文为墨色，但有极轻微的深浅变化

**朱批**：
- 页边（右侧）有朱红色批注
- 批注上方有一个小小的朱红"批"字印

**选择标记**：
- 正文中用户做过选择的地方，句末有一个朱红色的"·"
- 点击"·"，弹出小浮层，显示当时的选择记录

**印鉴**：
- 文末有一方"时光笺"印，朱红色，篆书
- 印色根据历史偏离度变化：正史偏朱红（#8B3E3C），野史偏赭石（#8B5E3C）

## 六、分享界面：缺角的船票

**故事卡正面**：
- AI生成的场景图（水墨淡彩+浮世绘版画风格）
- 标题
- 判词
- 右下角有物理的"缺角"

**故事卡背面**：
- 正文第一段
- 缺角处对应关键词图标

**合券交互**：
朋友收到故事卡后，在「时光笺」App中通过"扫一扫"或"相册导入"识别。如果朋友也拥有这张关键词卡，两张卡"合券"，双方获得一张限定的"合券纪念卡"。

## 七、动效与音效规范

### 动效原则
- **缓动曲线**：所有动画使用ease-out（慢出），符合"文雅"气质
- **时长**：一般过渡动画200-300ms，不超过400ms
- **涟漪**：墨滴落水的涟漪，使用弹性缓动，模拟水的质感

### 音效设计

| 场景 | 音效 |
|---|---|
| 抽卡·墨池涟漪 | 水滴滴落清水的"叮"声，极轻微 |
| 拂去墨迹 | 宣纸摩擦的沙沙声 |
| 卡片显现 | 轻微的风铃声 |
| 做选择手势 | 毛笔在砚台边舔笔的轻触声 |
| 章节结束涟漪 | 编钟单音，余韵悠长 |
| 关键词显灵 | 一声极轻的磬音 |
| 合券成功 | 两片玉器轻轻相击的清脆声 |

## 八、总结：视觉即故事

「时光笺」的UI美工设计，核心原则是：**每一个像素都在讲故事**。

- 墨池不是按钮，是一方用旧了的砚台
- 卡片不是数据，是刚写就的笺纸
- 卷轴不是阅读器，是说书人正在展开的书卷
- 分享不是链接，是撕下一角等人合券的船票

用户第一眼看到的，不是"一个App的界面"，而是"一间我可以走进去的书房"。
